Learn how to use this library

Main Goal of this library is to keep it as simple as possible. It's not really hard to implement an RSS Feed download by C#. My goal was to make it even simpler - especially for programming beginners.

How to install?

1. First: Download this library on Bitbucket and unzip

There are 2 Directories in the BIN-Directory you will find a "ready to use" pre-compiled DLL file. In the SRC-Directory you will find the current C# Solution - necessary if you want to compile your own Version.

For those who have created more features or fixed a bug. Please consider to commit and push your changes to the Bitbucket repository.

Link as a reference

Open your soluation explorer (View->Solution Explorer). In Solution Explorer click on "References -> Add Reference". Then click on "Search" and find the DLL file in the BIN-Directory.

In your Code-File, you need to add the namespace of the library:

            
                using wprssadpi;

Now you are ready to use this library.

Using the WordPressFeedController

The WordPressFeedController is a Singleton-Class that is only responsible to Load your RSS Feeds and serve an RSS Feed Articles. The Controller class has already implemented Get-Functions to simply Query your Wordpress Feed Articles. Of course it is also possible to query your articles by LINQ, because all articles are stored in a Generic List. 

            
                //Get an Instance of Wordpress Controller (Singleton)
                WordPressFeedController wp = WordPressFeedController.Instance;

                //Load all the RSS Articles
                wp.LoadRSS("http://www.its-all-about.de/rss");

                //Get the Newest Article (Check Docs for other functions)
                var rssItem = wp.GetNewestItem();

                this.label1.Text = rssItem.Title;

                //Text Only, Remove all the HTML Tags - Limit too 300 Chars
                this.richTextBox1.Text = wp.RemoveHTMLFromText(rssItem.Summary.Substring(0, 300)) + "...";

                //Open RSS Article on Button Click
                this.button1.Click += (s, e) => {
                    Process.Start(rssItem.Id);
                };
